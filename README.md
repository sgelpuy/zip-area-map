# 우편번호 구역 지도

우편번호(5자리, 국가기초구역 기준)를 여러 개 입력하면 카카오맵 위에 해당 구역 경계를 색깔별로 그려줍니다.

- 데이터 출처: 우정사업본부 "구역의도형" (`TL_KODIS_BAS`, 기초구역=현행 우편번호 경계), EPSG:5179 → WGS84 변환
- 전국 17개 시도, 34,516개 우편번호 구역 포함
- 순수 정적 사이트 (백엔드 서버 없음): `data/<앞2자리>/<우편번호>.json` 형태로 구역별 좌표를 잘라서 저장해두고, 입력한 우편번호만 그때그때 fetch

## 로컬에서 보기

카카오맵 JavaScript SDK는 등록된 도메인에서만 동작합니다 (localhost/127.0.0.1은 등록 불가). 로컬에서 지도 렌더링까지 확인하려면 실제 배포 도메인(GitHub Pages 등)을 카카오 디벨로퍼스에 등록해야 합니다.

```
python -m http.server 8080
```

## 데이터 갱신 / 지역 추가

1. `../구역의도형_전체분_<지역명>/` 폴더에 우정사업본부 구역의도형 shapefile을 압축 해제해서 넣기
2. `python convert.py` 실행 → `data/` 폴더가 갱신됨
3. 커밋 후 푸시하면 GitHub Pages에 자동 반영

## 카카오맵 도메인 등록

배포 후 나오는 GitHub Pages 주소(예: `https://<username>.github.io/<repo>/`)를 카카오 디벨로퍼스 콘솔 → 내 애플리케이션 → 플랫폼 → Web 플랫폼에 등록해야 지도가 정상적으로 뜹니다.
